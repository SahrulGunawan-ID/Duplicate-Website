# How To Run
````
apt update && apt upgrade -y
````

````
apt install golang-go -y
````

# Or 
````
apt install golang -y
````

````
git clone https://github.com/SahrulGunawan-ID/Duplicate-Website
````

````
cd Duplicate-Website
````

# For CPP
````
g++ webclone.cpp -o out_webclone
````

# Run cpp Files
````
./out_webclone
````
# For Golang
````
go run webclone.go
````
